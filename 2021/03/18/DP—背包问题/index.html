2. 完全背包

AcWing 完全背包问题 题源

  • $N$ 件物品，背包容量为 $V$ 。每件物品可使用无数次。

    第 $i$ 件物品的体积为 $v_i$，价值是 $w_i$。

    选择物品装入背包，使总体积不超过背包容量，且总价值最大。

  • 在01的基础上把
    dp[i][j] = max(dp[i][j], dp[i - 1][j - v[i]] + w[i])改为

    dp[i][j] = max(dp[i][j], dp[i][j - v[i]] + w[i])即可。

3. 多重背包

AcWing 多重背包问题1 题源

  • $N$ 件物品，背包容量为 $V$ 。

    第 $i$ 件物品的体积为 $v_i$，价值是 $w_i$，最多有$s_i$个。

    选择物品装入背包，使总体积不超过背包容量，且总价值最大。

  • 01背包的扩展，再加一层循环：k = 1, 2, 3, … s[i]表示选多少个：即

dp[j] = max(dp[j], dp[j - v[i]] + w[i])改为

dp[j] = max(dp[j], dp[j - v[i] * k] + w[i] * k)即可。
